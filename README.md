## Usage

1. add variable to Project > Settings > CI/CD > Variables

```
API_ACCESS_TOKEN=<your_access_token>
```

2. update gitlab-ci.yml

```
script:
    - yarn global add gitlab-comment
    - gitlab-comment "<whatever you want>"
```

## Run tests

```sh
npm run test
```
